## VitePress 한글 번역 저장소 입니다~ 😆

VitePress는 [Vite](https://vuepress.vuejs.org) 위에 구축된 [VuePress](https://github.com/vitejs/vite)의 정신적 후계자이며 Vue로 구동되는 정적 사이트 생성기입니다.

## 문서
